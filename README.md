# 🎮 Campus Zero: Reimagined

A multi-act 2D game built with **Phaser**, combining survival, combat, and chaotic platforming into one cohesive experience.

This project is being developed as a **team hackathon project**, with a strong focus on clean structure, collaboration, and beginner-friendly code.

---

## 🚀 Game Overview

The game is structured into multiple acts, each with a unique gameplay style:

### 🧩 Prologue
- Story-driven intro
- Typewriter-style dialogue system
- Sets the tone and context for the game

### 🧟 Act 1 — Survival (Among Us-style)
- Top-down gameplay
- Player is constantly hunted by a zombie
- Complete tasks while avoiding being caught
- Focus: tension, movement, and decision-making

### ⚔️ Act 2 — Combat
- Action-focused gameplay
- Fight enemies using combat mechanics
- Progress through encounters
- Focus: skill, timing, and strategy

### 😈 Act 3 — Level Devil
- Trap-based chaotic platforming
- Inspired by “Level Devil”
- Unexpected hazards and high difficulty
- Focus: reaction time and persistence

### 🏠 Landing Page
- Clean, professional UI
- Entry point into the game
- Designed to resemble a polished product
